# SOROEMONO

JetBrains Mono に対して 1:2 で合わせる日本語フォント。

- `v0.0.1` バグ: JetBrains Mono を先に指定しないとフォーカスポイントがズレる場合あり

## Build

```sh
$ brew install fontforge
$ fontforge -script build.py
```

## Font Family

```
'JetBrains Mono', SOROEMONO, ...
```
